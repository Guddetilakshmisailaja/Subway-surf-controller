# Subway-surf-controller
Gesture-Controlled Subway Surfers Game
Developed an interactive version of the popular endless runner game Subway Surfers using real-time hand gesture recognition.
Implemented computer vision techniques to detect and track hand movements for controlling player actions such as jump, slide, and lane switching.
Integrated gesture recognition with game logic to enable a touchless gaming experience, improving user engagement and accessibility.
Utilized technologies like Python, OpenCV, and MediaPipe for accurate and responsive gesture detection.
Optimized performance to ensure smooth gameplay with minimal latency in real-time interaction.
Demonstrated practical application of AI and computer vision in gaming interfaces.
